# BabyIM
Learn Instant Message (IM) like a baby playing

## 1、端到端全局

### 1.1、功能列表

- 账号
  - 注册 Register
  - 登陆 Login
  - 登出 Logout
- 联系人

- 会话类型
  - 单聊
  - 群聊
- 发消息
  - 发文字
  - 发文件
    - 发图片
    - 发语音
    - 发视频
